Information on data exporting

  • Data can be exported from multiple studies at once (e.g. sub-studies within a trial).
  • Remember to resolve duplicate replies for your studies before exporting if possible (get to this from the study edit page).
  • Also remember that duplicate replies will be exported, but marked 1 in the is_canonical_reply column.
  • To allow for a shared reference point (in time) for data exported, this form allows you to specify a 'reference study'.
  • The date the participant was randomised to the reference study is used to calculate how many days from baseline (day 0) each observation is. This may be helpful when participants can enter substudies at slightly differently latencies, but final analyses need and offset from a common reference date (e.g. treatment commencement)
